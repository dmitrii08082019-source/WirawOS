# ⚡ WirawOS — Свободная Операционная Система

**Версия:** v13.3  
**Размер ISO:** 3.26 ГБ  
**Сжатый 7Z:** ~1 ГБ  
**Требования к диску:** **4 ГБ+** (USB / HDD / SSD)  
**Цена:** Абсолютно БЕСПЛАТНО! 🎉  
**Лицензия:** Open Source (MIT)

---

## ⚠️ **ВАЖНО! ТРЕБОВАНИЯ К ДИСКУ!**

| Требование | Минимум |
|------------|---------|
| **Размер диска** | **4 ГБ+** |
| **Тип диска** | USB флешка / Жёсткий диск / SSD |
| **Файловая система** | FAT32 (для USB) / Любая (для HDD и SSD) |

> **❌ НЕЛЬЗЯ ИСПОЛЬЗОВАТЬ ДИСКИ МЕНЬШЕ 4 ГБ!**  
> Образ WirawOS весит 3.26 ГБ, поэтому диск должен быть **4 ГБ+**!

## 📦 **Что входит в WirawOS v13.3 (3.26 ГБ):**

| Компонент | Описание |
|-----------|----------|
| 🖥️ **Ядро** | Собственное ядро WirawOS |
| 🚀 **Загрузчик** | Быстрый загрузчик (boot.bin) |
| 🎨 **VGA драйвер** | Полноценный видео драйвер |
| 🛠️ **WirPE** | Preinstallation Environment |
| 📀 **Установщик** | Простой установщик системы |
| 📝 **Notepad** | Встроенный текстовый редактор |
| 🎮 **Игры** | Змейка, Тетрис, Сапёр |
| 🌐 **Сеть** | Поддержка TCP/IP |
| 🔊 **Звук** | Sound Blaster 16 драйвер |
| 📁 **Файловая система** | FAT16 + поддержка дисков |
| 💾 **Драйверы** | Для всех основных устройств |

## 🚀 **Как установить:**

### 1️⃣ **На реальный компьютер:**
```bash
1. Скачай WIRPE.7z (~1 ГБ)
2. Распакуй архив (пароль: 1)
3. Получи WIRPE.ISO (3.26 ГБ)
4. Подготовь USB флешку **4 ГБ+**
5. Запиши ISO на USB (UltraISO / Rufus)
6. Загрузись с USB
7. Следуй инструкциям установщика
