## 📦 **Что входит в WirawOS v13.3 (3.26 ГБ):**

| Компонент | Описание |
|-----------|----------|
| 🖥️ **Ядро** | Собственное ядро WirawOS |
| 🚀 **Загрузчик** | Быстрый загрузчик (boot.bin) |
| 🎨 **VGA драйвер** | Полноценный видео драйвер |
| 🛠️ **WirPE** | Preinstallation Environment |
| 📀 **Установщик** | Простой установщик системы |
| 📝 **Notepad** | Встроенный текстовый редактор |
| 🎮 **Игры** | Змейка, Тетрис, Сапёр |
| 🌐 **Сеть** | Поддержка TCP/IP |
| 🔊 **Звук** | Sound Blaster 16 драйвер |
| 📁 **Файловая система** | FAT16 + поддержка дисков |
